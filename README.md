# Rocket+CRM (RocketClients)

White-label GHL CRM reseller platform. Sell GoHighLevel as Rocket+CRM with integrated Rocket+ AI Mods.

## Live URLs

- **Marketing Site**: https://rocketclients.com
- **App (GHL SaaS)**: https://app.rocketclients.com

## Tech Stack

- **Framework**: Next.js 15
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Deployment**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

- `/` - Landing page
